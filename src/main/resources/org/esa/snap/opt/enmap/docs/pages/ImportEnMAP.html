The Environmental Mapping and Analysis Program (EnMAP) is a German hyper-spectral satellite mission to monitoring and characterise Earth?s environment on a global scale.

Sensor Characteristics
  • Imaging push-broom hyper-spectral sensor mainly based on modified existing or pre-developed technology
  • Broad spectral range from 420 nm to 1000 nm (VNIR) and from 900 nm to 2450 nm (SWIR) with high radiometric resolution and stability in both spectral ranges
  • 30 km swath width at high spatial resolution of 30 m x 30 m and off-nadir (30°) pointing feature for fast target revisit (4 days)

To open a product select in the file chooser either a file in the unzipped directory or directly the distributed zip file. Alternatively you can drag the files directly from the system file explorer on the Product Explorer in SNAP.
Supported EnMAP products are Level-1B, Level-1C and Level-2A.
Currently, only products provided in GeoTiff are supported.
